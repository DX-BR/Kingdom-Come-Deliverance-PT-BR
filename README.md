Tradução para Kingdom Come Deliverance PT-BR

# Como Instalar
1º Localize a pasta do seu jogo, e abra a pasta chamada Localization. [Imagem](https://ibb.co/sb2GsbW)

2º Faça uma copiar dos seguintes arquivos: English_xml.pak antes de fazer a instalação da tradução.

3º Extraia o arquivo winrar dentro da pasta.  

4º Subistituar o arquivo English_xml.pak na pasta Localization pelo arquivo
