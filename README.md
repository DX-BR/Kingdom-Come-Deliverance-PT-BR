# Kingdom-Come-Deliverance-PT-BR

Tradução para Kingdom Come Deliverance PT-BR

# Como Instalar
1º Localize a pasta do seu jogo, e abra a pasta chamada Localization. [Imagem](https://ibb.co/sb2GsbW)

2º Faça uma copiar do seguinte arquivo: English_xml.pak antes de fazer a instalação da tradução. [Imagem](https://ibb.co/fDT6bY1)

3º Subistituar o arquivo English_xml.pak [Imagem](https://ibb.co/Br8fNpd)

4º Bom jogo ! Agradeçemos por baixar nossa tradução feita por fans do jogo.
